# Kunal Umaji — personal page v1.0

A deliberately simple academic homepage inspired by traditional faculty and researcher websites.

## Preview

```bash
cd personal-page-v1.0
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Personalize

- Replace `assets/profile.jpg` with a square-cropped photograph when needed.
- Review the short introduction and research description.
- Add a Publications section only when there is something substantive to list.

The site uses plain HTML and CSS and has no build step.
